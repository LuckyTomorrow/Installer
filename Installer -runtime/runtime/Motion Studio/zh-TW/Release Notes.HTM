    What's new

    Motion Studio
    (V 1.6.0.2)

    New:

    1. 新增：MVP-3265-MAS板卡類型.
    2. 新增：Ethernet連接後，卸載虛擬軸卡.
    3. 新增：凸輪編輯器（CAM Editor）工具.
    4. 新增：支持Editor中宏定義名稱高亮顯示.
    5. 新增：氣油缸配寘時可配置別名（默認是Cylinder）.
    6. 修正：查找視窗打開時發佈工程會拋异常.
    7. 修正：SUB前面若有Static關鍵字，在Function List中沒有列出.
    8. 修正：發佈工程再次打開，發佈工程工具按鈕/工具選單還是使能的.
    9. 修正：重啓Runtime後，項目與控制器表單自動運行的task狀態不一致.
    10. 修正：modbus地址有衝突時，删掉衝突的部分，异常依然存在，導致不能保存.
    11. 優化：VR錶當Range中行數比較多（如8000）時，打開VR錶可能斷線.
      12.

    Motion Runtime
    (V 1.6.0.2)

    New:

    1. 新增：一次寫多個DO的modbus處理.
    2. 新增：PATHLINK_RDYPOINT，DORESET指令.
    3. 新增：latchbuffer相關指令：LBUF_RESET，LBUF_DATA，LBUF_STATUS等.
    4. 新增：Modbus Client指令：MB_OPEN，MB_CLOSE，MB_STATUS，MB_SETCOIL等.
    5. 新增：可根據群組速度改變PWM頻率或占空比的功能的相關指令：GPWM_LINKEN，GPWM_MODE，GPWM_REFVEL等.
    6. 修正：當Task中存在“|”時，下載檔案失敗.
    7. 修正：RUN_TASK指令啟動程式，有時會導致Task無法啟動，必須重啓Runtime才有效.
    8. 優化：沒有硬體設定時，仍然可以獲取到軸數.
    9. 優化：Reset axis error stop狀態的同時reset所在group的error stop state .
    10. 優化：RUN_TASK為了减少啟動時間，不再對原始程式碼進行編譯，需保證被啟動的task對應的exe已存在，否則報錯.
      11.
